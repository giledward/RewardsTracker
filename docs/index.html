How to read the page

This is a relationship board. The main question is simple: did BONK move more than SOL, less than SOL, or about the same?

Correlation: tells us how closely BONK and SOL moved together.
Performance spread: compares BONK’s percentage move against SOL’s percentage move.
Strength ratio: shows whether BONK is gaining or losing value compared with SOL.
