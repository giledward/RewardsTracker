How to read the page

This is a relationship board. The main question is simple: did BONK move more than SOL, less than SOL, or about the same?

Correlation: how closely BONK and SOL moved together.
Performance spread: BONK percentage move minus SOL percentage move.
Strength ratio: BONK price divided by SOL price.
